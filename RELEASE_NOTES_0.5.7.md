# NaxOne v0.5.7

## 性能优化

- **大幅降低 WMI 占用**：Windows 进程查询从 `tasklist.exe` / `wmic` 命令改为 `sysinfo` crate（走 NT 内核 API），完全绕开 WMI 接口。
  - WmiPrvSE.exe CPU：从 ~25% → 0%
  - 服务状态预热：从 ~10 秒 → 0.3 秒（快 35 倍）
  - 启动时不再有几秒钟卡顿
- **`get_app_stats` 加 2 秒兜底超时**：极端场景下进程枚举卡住时降级显示（内存列 "—"），不再阻塞前端 IPC。

## 体验改进

- **设置 → 应用更新**：新增检查更新面板。圆形蓝按钮点击有旋转动画，发现新版本时会显示红点提醒和「点击更新」入口（直接 OTA 安装、无需手动下载）。
- **活动日志面板**：背景从透明白调整为 88% 不透明 + 玻璃模糊，文字更易读、不再透出花花绿绿的光晕。

## 开发体验

- **dev 与正式版可并行运行**：debug build 不再注册 single-instance plugin，方便边开发边查看正式环境。两边共享配置目录，知道写入会有竞态即可。

## 升级方式

- v0.5.6 用户：在设置 → 应用更新点圆形按钮，或直接保持仪表盘 banner 提示。
- v0.5.5 及更早用户：手动下载本页 `NaxOne_0.5.7_x64-setup.exe` 覆盖安装一次。
